Format und Groesse:

- Jörg: SVG mit ViewBox 280x360, PNG in 560x720

Einsetzen:
1. Bestehenden Ordner assets/images im Spiel sichern.
2. Diese Dateien in assets/images kopieren und vorhandene Dateien ueberschreiben.
3. GitHub aktualisieren und im Browser hart neu laden (Strg+F5).
